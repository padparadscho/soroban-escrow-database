# Soroban Escrow Database

Shared database for the [soroban-escrow-notifier](https://github.com/padparadscho/soroban-escrow-notifier) and [soroban-escrow-dashboard](https://github.com/padparadscho/soroban-escrow-dashboard) projects, providing a single source of truth for migrations against the shared Neon PostgreSQL instance.
